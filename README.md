C Programming: A Modern Approach – Solutions

This repository contains my personal solutions and practice work for exercises from C Programming: A Modern Approach by K. N. King.

The main goal of this repository is to document my learning process while improving my understanding of the C programming language, problem-solving, and low-level programming concepts.

About the Repository

While working through the book, I solved exercises and programming projects chapter by chapter.
The solutions in this repository are written by me and reflect my own approach to understanding and applying the concepts introduced in the book.

This repository is intended as a learning archive and portfolio project, showing my progress in C programming through consistent practice.

What I Practiced
Basic C syntax and program structure
Variables, data types, and operators
Control flow with conditions and loops
Functions and modular programming
Arrays and strings
Pointers and memory-related concepts
Input and output handling
Structs, unions, and enums
Preprocessor usage
Problem-solving with C
Repository Structure

The repository may be organized by chapters, for example:

Chapter1/
Chapter2/
Chapter3/
...

Each folder contains my solutions for the corresponding chapter exercises or programming projects.

Most solutions can be compiled with a C compiler such as gcc:

gcc filename.c -o prog
./prog

For projects that include a Makefile, run:

make

Disclaimer

These are my personal solutions and are not official solutions to the book.
They are shared for educational and portfolio purposes only.
